# MailMate AI

MailMate AI is a web application developed for the INFS3203 team project.  
The app helps users generate email replies using AI. A user can paste an incoming email, choose the tone and reply length, and get a ready-to-send reply.

## Main Features
- Generate AI email replies
- Choose tone (formal, friendly, professional)
- Choose reply length (short, medium, detailed)
- Improve generated reply
- Save replies in the database
- View saved reply history

## Tech Stack
- Backend: Python Flask
- AI: Gemini API
- Database: SQLite
- Containerization: Docker
- CI/CD: GitHub Actions
- Deployment: Render
