# MailMate AI

MailMate AI is a web application for the INFS3203 team project.  
It helps users generate email replies using AI. The user pastes an incoming email, chooses the tone and reply length, and gets a ready-to-send reply.

## Main Features
- Generate AI email replies
- Choose tone
- Choose reply length
- Improve generated reply
- Save replies in the database
- View saved reply history

## Tech Stack
- Python Flask
- Gemini API
- SQLite
- Docker
- GitHub Actions
- Render
